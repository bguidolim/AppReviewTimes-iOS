# AppReviewTimes-iOS
An app to check the average review times for iOS and Mac apps.

#Run
 - ```git clone https://github.com/bguidolim/AppReviewTimes-iOS.git AppReviewTimes-iOS```
 - ```pod install```
 - Be happy :)
